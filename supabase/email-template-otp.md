# メールテンプレート（確認コード用）

アプリは **メール内の確認コード**（多くの場合 **8桁**）でログインします。  
Supabase のメールに `{{ .Token }}` が含まれている必要があります。

## 設定手順

1. [Supabase Dashboard](https://supabase.com/dashboard) → プロジェクト
2. **Authentication** → **Email Templates**
3. **Magic Link** を開く
4. 本文に **確認コード** を表示するよう編集

## 本文の例（日本語）

**Subject:** `家計簿 ログインコード`

**Body:**

```
家計簿へのログイン

次の確認コードをアプリの画面に入力してください。

{{ .Token }}

このコードは短い時間だけ有効です。
心当たりがない場合はこのメールを無視してください。
```

## 注意

- `{{ .Token }}` は Supabase が自動で置き換える数字です（**必須**）
- 桁数は Supabase の設定により **6桁または8桁** などになることがあります（本アプリは **8桁** 想定）
- リンク（`{{ .ConfirmationURL }}`）は不要なら削除してよい
