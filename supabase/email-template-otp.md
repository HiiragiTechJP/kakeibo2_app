# メールテンプレート（6桁コード用）

アプリは **メール内の6桁コード** でログインします。  
Supabase のメールに `{{ .Token }}` が含まれている必要があります。

## 設定手順

1. [Supabase Dashboard](https://supabase.com/dashboard) → プロジェクト
2. **Authentication** → **Email Templates**
3. **Magic Link**（または Confirm signup）を開く
4. 本文に **6桁コード** を表示するよう編集

## 本文の例（日本語）

**Subject:** `家計簿 ログインコード`

**Body（HTML でも Plain でも可）:**

```
家計簿へのログイン

次の6桁コードをアプリの画面に入力してください。

{{ .Token }}

このコードは短い時間だけ有効です。
心当たりがない場合はこのメールを無視してください。
```

## 注意

- `{{ .Token }}` は Supabase が自動で置き換える **6桁の数字** です（必須）
- リンクだけのテンプレートのままだと、ユーザーがコードを見つけられません
- メールにリンク（`{{ .ConfirmationURL }}`）だけがある場合も、**コード入力はアプリ側で行う** のでリンクは押さなくて大丈夫です（リンクは削除しても可）
